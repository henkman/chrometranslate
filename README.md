chrometranslate
===============

Translate selected text immediatly

[Download(crx)](https://github.com/papplampe/chrometranslate/raw/master/dist/chrometranslate.crx)


Installation:

1. make new tab with chrome://extensions/
2. drag and drop the crx into it
